# Node.js Passport Authentication
---
### 技术栈
* Node.js
* Express
* mongodb+mongoose
* ejs
* passport
* Express-session

### Usage
```sh
  $ npm install
```

```sh
  $npm start
  # Or run with Nodemon
  $ npm run dev

  #visit http://localhost:5000
```

### MongoDB
> 打开 "config/keys.js 配置修改MongoDB URI , 本地或者线上地址"
